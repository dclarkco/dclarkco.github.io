Version 1

I decided to use Altair (big mistake), as I was somewhat familiar with it from INFO 4602, and I prefer python data manipulation with pandas instead of javascript. That said, getting Altair charts to work in an existing html document was pretty much impossible, so I had to link to basic html files for each vis. Altair docs suck, tell your friends. For my data, I used the FiveThirtyEight 2018 March Madness data, which is a nice mostly clean dataset with simple values. I focused on the mens league, and cleaned out some wierd seed data and other metadata stuff with pandas. This was the easy part. Then, I slapped it into an Altair template and called it.

For version 2, I cleaned up the Y axis and removed some more bad data, as well as cleaning up the chart a bit. I also expanded the viewport for the chart to a meager 1000x600 to make the data more visible.
