Redesign

  1. Home
  2. Redesign

While the template provided most of the bootstrap methods I'd be using, I needed to understand how they worked. In particular, I wanted the vertical navbar that scrolls down and automatically highlights the current viewport section on the bar. That is the largest reason I picked this template, which uses a combination of .flex-column and d-flex to make flexible, vertical nav bars. Then, bootlegging some of the javascript from the template, the current section is highlighted as it is scrolled through. This was the major feature I was interested in implementing, so finding a template with the javascript done already was a big step for the redesign. I also used some breadcrumbs, and a lot of containers.

About Me:

The about me section was fairly straightforward, I added some key information, like my degree, email, website, location, and pronouns, as well as a redone blurb about myself.

Resume:

My resume is already in pdf format, so I just stuck the pdf iframe into the Resume container and called it a day. Some formatting work had to be done, but the pdf iframe is very useful as is. I used the skills container from the template to showcase some of my skills.

Portfolio:

My portfolio isn't as fleshed out as I'd like it to be, so I simplified the lightbox containers to simply show the image in full screen when clicked on, as opposed to linking to a separate page. This uses some javascript from bootstrap and elsewhere.

Contact:

This section was also in the template, albeit with a form submission I didn't want to use. I wiped that out, added a linkedin link, a mailto: email link, and changed the google maps iframe to show Denver. This uses some flex columns to showcase all the information.
